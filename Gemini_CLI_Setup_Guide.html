Gemini CLI Setup Guide

v0.24.0

1 Obtain a Google AI API Key

To power the CLI, you need an API key from Google AI Studio.

  • Visit Google AI Studio.
  • Click "Create API key".
  • Copy the key and store it securely.

2 Configure Environment Variables

The Gemini CLI looks for your key in the GOOGLE_API_KEY environment variable.

Add this to your .zshrc or .bash_profile:

export GOOGLE_API_KEY="your_api_key_here"

Then reload your terminal: source ~/.zshrc

3 Install Moodle MCP Integration

Connect your Moodle tool to the Gemini CLI so you can ask about your courses and gradebooks.

Run the following command (update paths as needed):

gemini mcp add moodle python -m moodle_mcp.main --env PYTHONPATH="/absolute/path/to/MoodleAPI/src"

4 Core CLI Commands

The CLI can be used in several modes:

Interactive Mode: Just type gemini to start a chat session.

One-Shot Prompt: gemini "Tell me about course ID 6292"

YOLO Mode: gemini -y (Automatically accepts all tool calls without promptingâ€”use with caution!)

5 Troubleshooting

  • Command not found: Ensure your Python scripts folder is in your system PATH.
  • Permission Denied: Ensure you have granted "Web Service" permissions to your token in Moodle.
  • MCP Server Error: Check that your PYTHONPATH accurately points to the src directory of the MoodleAPI project.
